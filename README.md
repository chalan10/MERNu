# Overview
MERNu is a web application utilizing the MERN stack that provides a food ordering platform for restaurants and customers.

# User Functionality
### Restaurants
* Account Page
* Manage menu

### Customers
* Account Page
* Place orders

# Technologies Used
* MongoDB + Mongoose
* Express
* React
* Node
* Passport + JWT Authentication

# Future Plans
### Restaurants
* Manage active orders
* Order History
* Daily/Weekly/Monthly Overview
* Item Modifiers
* Delivery

### Customers
* Cancel active order
* Order History
* Order again
